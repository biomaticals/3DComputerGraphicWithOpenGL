# 오픈지엘로 배우는 3차원 컴퓨터 그래픽

## 개요

이 프로젝트는 OpenGL API를 체계적으로 학습하고 실습하기 위해 제작한 학습용 프로젝트입니다.
주우석 저자의 [오픈지엘로 배우는 3차원 컴퓨터 그래픽스]를 기반으로 하되, 
각 예제를 최신 라이브러리를 활용하여 재구성했습니다.
이 학습 도구를 통해 이론, 코드, 실행 결과까지 확인할 수 있습니다.
"선택 창"에서 파트, 챕터, 코드를 선택하면, "입력 창"에 입력 코드를 표시합니다.
"설명 창"에서는 주요 함수들과 이론을 설명하고 디버그 정보를 보여주고, "결과 창"에서 결과를 확인합니다.
시연 영상은 아래의 개발 일기 링크에서 확인할 수 있습니다.

책에 대해 더 알고 싶다면 아래 링크를 참고합니다.
https://www.hanbit.co.kr/store/books/look.php?p_code=B1779572378

이 프로그램의 오픈소스는 아래 링크에서 확인할 수 있습니다.
https://github.com/biomaticals/3DComputerGraphicWithOpenGL

## 실행 방법 - exe
1. 깃에서 레포지터리를 다운받습니다.
2. 3DComputerGraphicWithOpenGL/3DComputerGraphicWithOpenGL/Binaries로 이동합니다.
3. 본인의 환경에 맞는 플랫폼과 구성을 찾아 안으로 이동하여 3DComputerGraphicWithOpenGL.exe를 실행합니다.

## 실행 방법 - debug
1. 깃에서 레포지터리를 다운받습니다.
2. 3DComputerGraphicWithOpenGL로 이동합니다.
3. 3DComputerGraphicWithOpenGL.sln을 실행합니다.
4. 3DComputerGraphicWithOpenGL 프로젝트를 시작 프로젝트로 설정하고 디버깅을 시작합니다.

# 3D Computer Graphics with OpenGL

## Overview

This project is a learning tool designed to systematically study and practice the OpenGL API.  
It is based on the book "3D Computer Graphics with OpenGL" by Woo-Seok Joo,  
and each example has been reconstructed using modern libraries and tools.

With this learning environment, you can explore theory, examine source code,  
and immediately view execution results.  
By selecting a part, chapter, or example in the "Selection Panel",  
the corresponding source code is displayed in the "Code Panel".  
The "Description Panel" provides explanations of key functions, theoretical notes,  
and debugging information, while the "Result Panel" shows the rendered output.
The demonstration video is available at the development log link below.

For more information about the book, please refer to the link below:  
https://www.hanbit.co.kr/store/books/look.php?p_code=B1779572378

The full source code of this program is available on GitHub:  
https://github.com/biomaticals/3DComputerGraphicWithOpenGL

## How to Run – Executable
1. Download the repository from GitHub.  
2. Navigate to `3DComputerGraphicWithOpenGL/3DComputerGraphicWithOpenGL/Binaries`.  
3. Choose the folder that matches your platform and configuration,  
   then run `3DComputerGraphicWithOpenGL.exe`.

## How to Run – Debug Mode
1. Download the repository from GitHub.  
2. Navigate to the `3DComputerGraphicWithOpenGL` directory.  
3. Open `3DComputerGraphicWithOpenGL.sln`.  
4. Set the `3DComputerGraphicWithOpenGL` project as the startup project  
   and begin debugging.

## Tech Stack
API & Libraries: OpenGL 4.6, ImGui, GLFW, GLM, glad, stb_image, TinyObjectLoader, etc.  
IDE: Visual Studio 2023  
Version Control: GitHub, Fork
